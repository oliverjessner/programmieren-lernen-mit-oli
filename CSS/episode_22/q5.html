Q5: Was ist der Unterschied zwischen Pseudo-Klassen und Pseudo-Elemente?

A:

  • : Pseudo-Klasse
  • :: Pseudo-Element

Eine CSS-Pseudo-Klasse ist ein Schlüsselwort, das einem Selektor hinzugefügt wird und einen speziellen Zustand des ausgewählten Elements angibt. Zum Beispiel die Pseudo-Klasse :hover.

Pseudo-Elemente verhalten sich ähnlich. Allerdings wirken sie so, als hätten Sie ein völlig neues HTML-Element zur Auszeichnung hinzugefügt. 

 
    div:hover {
        background-color: red;
    }

    div::after {
        content: "Pseudo Elemente";
    }
