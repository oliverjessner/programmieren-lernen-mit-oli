Q2: Was ist der universal selector?

A:

Der universal selector ist ein CSS selector, der auf alle Elemente einer HTML Seite angewendet wird anstatt nur auf ein bestimmtes Element. Er wird mit einem Sternchen (*) dargestellt. 

 
    * {    
        padding: 0;
        margin: 0;  
        box-sizing: border-box;
     }
