Q9: Nenne alle Selectoren die du kennst

A:

 
    /* Universal Selector */
    * {
        margin: 0;
        padding: 0;
    }

    /* Type Selector oder Element Selector */
    h1 {
        color: red;
    }

    /* Grouping Selector */
    h1, h2, h3 {
        color: white;
    }

    /* ID Selector */
    #id {
        color: green;
    }


    /* Class Selector */
    .class {
        color: blue;
    }

    /* Attribute Selector */
    input[type="text"] {
        color: yellow;
    }

    /* Child combinator */
    div > p {
        color: orange;
    }

    /* Adjacent sibling combinator */
    div + p {
        color: purple;
    }

    /* General sibling combinator */
    div ~ p {
        color: pink;
    }
