Q3: Erkläre das Box Model

A:

Das CSS-Box-Modell definiert das Layout und das Design von CSS-Elementen. Die Elemente sind der Inhalt (wie Text und Bilder), Padding (der Bereich um den Inhalt), Border (der Bereich um das Padding) und Margin (der Bereich um die Border).
