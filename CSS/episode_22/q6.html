Q6: Schreibe einen Selector für das zweite Input Element.

Dieser Selector darf nur das zweite Input Element verändern, du darfst nth-child und andere pseudo klassen nicht verwenden. Stelle dir vor, du hast keinen Zugriff auf den HTML Code. Das Element soll eine rote Hintergrundfarbe bekommen. 

 
    <input type="text" />
    <input type="number" />
    <input type="email" />

A:

Attribute selectors

 
    input[type="number"] {
        background-color: red;
    }
