Q9: Was is eigentlich HTML?

A:

HTML (Hypertext Markup Language) ist eine Auszeichnungssprache, die Webbrowser darüber informiert, wie die von Ihnen besuchten Webseiten strukturiert werden sollen. Sie kann komplex oder einfach gestaltet sein. HTML besteht aus einer Reihe von Elementen, um verschiedene Teile des Inhalts einzuschließen, zu umhüllen oder zu markieren, um ihn auf eine bestimmte Weise darstellen oder funktionieren zu lassen. Tags können Inhalte in einen Hyperlink verwandeln, um eine Verbindung zu einer anderen Seite herzustellen, Wörter kursiv darstellen usw.
